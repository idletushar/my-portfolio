TextUtils

This project is a React-based text manipulation app that enables users to modify text in various ways. It provides a user interface where users can input text and perform operations such as converting it to lowercase, uppercase.

News App

This project is a React-based news app that fetches and displays news articles from diverse sources and categories using the NewsAPI. React powers the user interface, state management, and article rendering. Users can browse categories, and explore detailed articles with additional information and images.

My NoyeBook

This project is a React-based notebook application that enables users to store and manage their notes. It provides a user interface where users can create, edit, and delete notes, as well as organize them into different categories or tags
